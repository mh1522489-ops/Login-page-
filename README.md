# 📁 Sarah's Portfolio | Interactive Login Interface

## 🧠 Project Overview
A modern, fully responsive login interface and portfolio navigation system built with pure HTML5 and CSS3. The project features custom micro-interactions, smooth transitions, and a visually engaging gradient-based design that enhances user experience and interface accessibility.

---

## 🛠️ Technologies Used
- **HTML5** (Semantic structure, forms, input validation)
- **CSS3** (Flexbox, Gradients, Transitions, Transforms, Media Queries)

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| **Custom Hover Effects** | Navbar links scale and change color smoothly on hover |
| **Interactive Form Fields** | Input fields rotate slightly on hover and glow with a gold shadow on focus |
| **Animated Call-to-Action** | Gradient button (gold/orange) with hover scale effect and active press feedback |
| **Glassmorphism Card** | Semi-transparent background with backdrop-blur for modern aesthetic |
| **Fully Responsive** | Mobile-first design with media queries for all screen sizes |
| **Focus States** | Enhanced accessibility with visible focus indicators on all interactive elements |
| **Smooth Transitions** | Consistent `0.3s ease` transitions across all UI components |

---

## 🎯 Unique Design Touches
- `transform: rotate(1deg)` on input hover — a subtle, playful micro-interaction
- Dual-layer gradients (body + card container) for depth
- Underline hover effects on "Forgot Password" and "Sign Up" links with color shift

---

## 📱 Responsive Breakpoints
- **Desktop**: Full layout with max-width 450px for form card
- **Mobile (≤600px)**: Reduced padding, smaller fonts, adjusted navbar spacing
